# ДЗ№1 по курсу Kotlin lev.1
Что сделано:
1. Реализована функция NumbersGrid(), которая отображает индексы исходного списка
2. Используется NumberViewModel, т.к. при смене ориентации экрана remember не запоминает значения списка
3. Каждый элемент списка отображается через NumberItem, в котором заданы параметры (цвет - красный для четных и синий для нечетных индексов - и форма квадарата) в зависимости от отображаемого индекса
4. Условие if (config.orientation == Configuration.ORIENTATION_PORTRAIT) 3 else 4 изменяет количество квадратов в одном ряду
5. Все элементы отображаются через LazyVerticalGrid
6. aspectRatio() помогает адаптировать размер квадрата, не растягивая его форму
   
Важно:
1. персистентное хранилище не используется
2. uses-permission INTERNET в манифесте отсутствует
3. используемые ресурсы описаны в файлах strings, colors и dimens
4. элементы сохраняются при повороте экрана
